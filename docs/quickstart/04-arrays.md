# Arrays

```ruby
fruits = ["apple", "banana", "cherry"]
puts fruits[0]        # apple
puts len(fruits)      # 3
```

## Append

```ruby
fruits = append(fruits, "date")
puts len(fruits)      # 4
```

## Index Assignment

```ruby
fruits[1] = "blueberry"
puts fruits[1]        # blueberry
```

## Nested Arrays

```ruby
matrix = [[1, 2], [3, 4]]
puts matrix[0]        # [1, 2]
```

## Iterating

```ruby
for fruit in fruits
  puts fruit
end
```

See [For Loops](07-for-loops.md) for more iteration patterns.
